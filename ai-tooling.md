# AI Tooling

## Tools Used

### OpenAI
Used for:
- Some of the code generation
- Debugging errors
- Designing system architecture

### GitHub Copilot
Used for:
- Code suggestions
- Faster development

---

## What Worked Well

- Rapid prototyping of RAG system
- Assistance with debugging deployment issues
- Helped structure project architecture

---

## Challenges

- Some generated code required adjustments
- Deployment troubleshooting required manual fixes
- AI suggestions sometimes outdated (e.g., LangChain changes)

---

## Conclusion

AI tools significantly improved development speed and problem-solving but required developer validation and adjustments.
